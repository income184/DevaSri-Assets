# Amma Sevai Maiyam (அம்மா சேவை மையம்)

இந்த repository-ல் native Android application (Kotlin + XML) உருவாக்கப்பட்டுள்ளது.

## முக்கிய அம்சங்கள்
- பயன்பாட்டு தலைப்பு: **அம்மா சேவை மையம்**
- தமிழில் மட்டும் UI
- தலைப்பை அடிப்படையாகக் கொண்ட சுலோகம்: **"மக்களுக்கு உதவுவது நம் முதன்மை கடமை"**

## திட்ட அமைப்பு
- `app/src/main/java/com/ammaservice/maiyam/MainActivity.kt`
- `app/src/main/res/layout/activity_main.xml`
- `app/src/main/res/values/strings.xml`

## இயக்குவது எப்படி
1. Android Studio-வில் project-ஐ திறக்கவும்.
2. Gradle sync செய்யவும்.
3. Emulator அல்லது Android device-ல் run செய்யவும்.
